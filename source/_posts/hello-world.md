## git 推送代码到github

```bash
    # 第一次提交
    git init
    git add README.md
    git commit -m "first commit"
    git branch -M main
    git remote add origin https://github.com/wuruixuan2464/personal.git
    git push -u origin main

    # 推送
    git add .
    git commit -m '提交说明'
    git push
```
